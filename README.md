# 📋 Registration Management System (RMS)

A modern **Registration Management System (RMS)** developed with **PHP, MySQL, HTML5, CSS3, JavaScript, and Git**.

The application allows administrators to securely manage registration records through a clean and professional dashboard.

---

## 🚀 Features

- 🔐 Secure administrator login
- 📝 Register new records
- 👀 View all submissions
- ✏️ Edit existing records
- 🗑️ Delete records
- 🔍 Search records
- 📊 Dashboard statistics
- 🎨 Responsive professional interface
- 💾 MySQL database integration
- 🔒 Secure database operations using PDO

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/zahabhh630-ops/registration-management-system-_RMS_-.git
```

2. Move the project into your XAMPP `htdocs` directory.

3. Import the MySQL database.

4. Start Apache and MySQL using XAMPP.

5. Open your browser and visit:

```
http://localhost/RMS/
```
